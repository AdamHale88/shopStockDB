## Installation

<p>To get started we must choose out datatbase with Mysql.</p>

```mysql
mysql -u -root -p
```
Then we must tpye

```mysql
SOURCE db/schema.sql
```
This selects our Database we wish to use.

```node
npm run seed
```
This will seed out database with data.

```node
npm start
```
This initiates the server for us to query
